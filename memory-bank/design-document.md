# Project Seer（赛尔号复刻版）

## 游戏设计文档（Game Design Document, GDD）

---

## 1. 游戏概述（Overview）

- **游戏类型**：回合制角色扮演 / 精灵养成 / 网页游戏风格  
- **核心循环**：  
  接取任务 → 探索星球 → 捕捉 / 对战精灵 → 养成强化 → 挑战高难副本
- **目标受众**：怀旧玩家

---

## 2. 核心系统机制（Core Mechanics）

### 2.1 精灵系统（Elf System）

#### 基础数值
- 体力（HP）
- 攻击（ATK）
- 特攻（SP.ATK）
- 防御（DEF）
- 特防（SP.DEF）
- 速度（SPD）

#### 属性体系
- 水 / 火 / 草 / 电 / 机械 / 冰 / 飞行 / 普通 / 超能 / 幽灵 / 地面
- 属性克制关系：
  - 当攻击方对于防御方的关系为“克制”，造成2倍伤害。
  - 当攻击方对于防御方的关系为“普通”，造成1倍伤害。
  - 当攻击方对于防御方的关系为“微弱”，造成0.5倍伤害。
  - 当攻击方对于防御方的关系为“无效”，造成0倍伤害。
## 草
- 克制：水、地面、光  
- 微弱：草、火、飞行、机械、圣灵
- 无效：无  

## 水
- 克制：火、地面  
- 微弱：草、水、圣灵
- 无效：无  

## 火
- 克制：草、机械、水
- 微弱：冰、火、圣灵
- 无效：无  

## 飞行
- 克制：草、战斗
- 微弱：电、冰  
- 无效：无  

## 电
- 克制：水、飞行、暗影
- 微弱：地面、神秘、圣灵
- 无效：地面  

## 机械
- 克制：冰、战斗
- 微弱：地面、火、战斗
- 无效：无  

## 地面
- 克制：火、电、机械
- 微弱：草、水、冰
- 无效：飞行  

## 普通
- 克制：无  
- 微弱：无  
- 无效：无  

## 冰
- 克制：草、飞行、地面
- 微弱：火、机械、战斗、龙、圣灵
- 无效：无  

## 超能
- 克制：战斗、神秘
- 微弱：光、暗影
- 无效：光  

## 战斗
- 克制：机械、冰、龙、圣灵  
- 微弱：超能、战斗、暗影
- 无效：无  

## 光
- 克制：超能、暗影
- 微弱：机械、冰、光、圣灵
- 无效：草  

## 暗影
- 克制：超能、暗影
- 微弱：机械、冰、光、圣灵
- 无效：无  

## 神秘
- 克制：电、神秘、圣灵
- 微弱：地面、战斗
- 无效：无  

## 龙
- 克制：冰、龙、圣灵 
- 微弱：草、水、火、电
- 无效：无  

## 圣灵
- 克制：草、水、火、电、冰
- 微弱：战斗、神秘、龙
- 无效：无  



#### 养成路径
- **升级**：通过对战获取经验值
- **进化**：
  - 达到特定等级
  - 或使用指定进化道具
  - 操作地点：实验室
- **融合**：
  - 两只指定精灵合成为新精灵
  - 操作地点：实验室

#### 捕捉机制
1. 进入野外战斗
2. 削弱野生精灵 HP
3. 投掷捕捉胶囊
4. 根据概率判定成功 / 失败

---

### 2.2 战斗系统（Battle System）

- **模式**：1v1 回合制战斗
- **流程**：
  1. 双方选择技能或使用物品
  2. 比较速度值（SPD），决定出手顺序
  3. 计算伤害
  4. 结算 HP

- **伤害计算要素**：
  - 攻击 / 防御差值
  - 技能威力
  - 属性克制系数

- **失败条件**：精灵 HP 归零即战斗不能

---

### 2.3 任务系统（Quest System）

#### 主线 / 基础任务
- **接取地点**：船长室
- **任务内容**：
  - 剧情对话
  - 解锁新地图
  - 击败指定 BOSS

#### 每日任务
- **刷新时间**：每日 00:00
- **任务类型**：
  - 资源采集
  - 击败指定数量野生精灵
- **奖励**：
  - 赛尔豆
  - 经验券

## 3. 游戏世界与场景设计（World & Level Design）

### 3.1 玩家大本营：赛尔号飞船（Seer Spaceship）

作为核心 HUB，连接所有系统模块。

| 场景名称 | 核心功能 | 交互逻辑 |
|--------|----------|----------|
| 船长室 | 任务中心 | 接取 / 提交主线任务，查看剧情进度 |
| 机械室 | 商店 / 装备 | 使用赛尔豆购买胶囊、药剂、装备 |
| 实验室 | 养成中心 | 精灵数值分配、进化、融合 |
| 传送舱 | 导航 / 图鉴 | 星系切换、精灵图鉴查看 |
| 能源中心 | 资源转化 | 捐献矿石 / 能源换取赛尔豆 |
| 资料室 | 资讯 / 活动 | 历史剧情、往期活动展示、激活兑换码 |

### 3.3 挑战区域：太空站（Space Station）

PvE 高难玩法集中地。

#### 勇者之塔
- 60 层闯关，难度递增
- 已通关层数可每日快速扫荡 2 次
- 隐藏内容：
  - 神秘领域
  - 谱尼挑战（七道封印 + 真身）

#### 命运之轮
- RNG 爬塔玩法
- 抽牌决定事件：
  - 上升 / 下降层数
  - 触发战斗
- 目标：抵达顶层击败 BOSS

#### 暗黑武斗场
- 进入条件：特殊门票 / 限制条件
- 奖励：暗黑版异色精灵

## 4. 开发路线规划（Roadmap）

### Phase 1：最小可行性产品（MVP）
- 战斗引擎
- 一只初始精灵
- 船长室（任务系统）

### Phase 3：功能完善
- 实验室系统（进化）
- 机械室系统（商店）

### Phase 4：高难玩法
- 太空站开放
- 勇者之塔
- 命运之轮
